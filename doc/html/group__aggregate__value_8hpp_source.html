8 /**
9  * Describes the intermediate state as well as the computation (aggregation,
10  * combining and output) for an aggregation operation.
11  *
12  * Essentially, the group_aggregate_value must describe a parallel reduction in
13  * the following form:
14  *
15  * \code
16  * for streamid in data_stream:
17  * for data in stream[streamid]:
18  * value[streamid].add_element(data)
19  * value[streamid].partial_finalize()
20  *
21  * for streamid in data_stream:
22  * final_value.combine(value[streamid])
23  *
24  * output = final_value.emit()
25  * \endcode
26  *
27  * Each value could have an arbitrary number of elements inserted into it.
28  * When all insertions are complete, partial_finalize() is called on the value.
29  * Values can be combined with each other to form a final value, which then
30  * emits a response.
31  */
32 class group_aggregate_value {
