14  * Describes a function capture closure.
15  *
16  * Defines a closure class describing a lambda closure. Contains 2 fields:
17  *
18  * \param native_fn_name The toolkit native function name
19  *
20  * \param arguments An array of the same length as the toolkit native function.
21  * Each array element is an array of 2 elements [is_capture, value]
22  * \code{.py}
23  * If is_capture == 1:
24  * value contains the captured value
25  * If is_capture == 0:
26  * value contains a number denoting the lambda argument position.
27  * \endcode
28  *
29  * Example:
30  * \code{.py}
31  * lambda x, y: fn(10, x, x, y)
32  * \endcode
33  *
34  * Then arguments will be
35  *
36  * [1, 10], --> is captured value. has value 10
37  * [0, 0], --> is not captured value. is argument 0 of the lambda.
38  * [0, 0], --> is not captured value. is argument 0 of the lambda.
39  * [0, 1] --> is not captured value. is argument 1 of the lambda.
40  * \endcode
41  */
